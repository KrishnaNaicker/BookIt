# BookIt: Experiences & Slots

A fullstack travel experience booking platform built with React, TypeScript, Node.js, and PostgreSQL.

## Project Structure

```
bookit/
├── frontend/          # React + TypeScript + Vite
├── backend/           # Node.js + Express + PostgreSQL
└── README.md
```

## Getting Started

See individual README files in `frontend/` and `backend/` directories for setup instructions.

## Tech Stack

**Frontend:**
- React 18
- TypeScript
- Vite
- TailwindCSS
- React Router
- Axios

**Backend:**
- Node.js
- Express
- TypeScript
- PostgreSQL
- CORS

## Development Progress

This project is being built step-by-step following a comprehensive learning guide.
